VDO.Ninja Integration

Optimize your VDO.Ninja streaming setup with Electron Capture.

Why Use Electron Capture with VDO.Ninja?

Electron Capture was originally built for VDO.Ninja and offers several advantages over OBS Browser Source:

  • No video smearing or desync issues
  • Lower CPU usage with hardware encoding
  • Auto-select screens without user prompts
  • Global hotkeys for muting (CTRL+M)
  • Latest Chromium with cutting-edge WebRTC features
  • Won't crash if OBS crashes

Basic VDO.Ninja Setup

View a Guest

elecap.exe --width=1920 --height=1080 --url="https://vdo.ninja/?view=abc123&scene" --title="Guest 1"

View Multiple Guests

# Scene link with all guests
elecap.exe --url="https://vdo.ninja/?scene&room=myroom" --title="All Guests"

# Individual guest windows
elecap.exe -w 640 -h 360 -x 0 -y 0 --url="https://vdo.ninja/?view=guest1&room=myroom" -t="Guest 1"
elecap.exe -w 640 -h 360 -x 640 -y 0 --url="https://vdo.ninja/?view=guest2&room=myroom" -t="Guest 2"

Screen Sharing with VDO.Ninja

Auto-start screen sharing without user interaction:

# Share primary display
elecap.exe --node 1 --url="https://vdo.ninja/?push=mystream&ss=1&autostart"

# Share a specific window
elecap.exe --node 1 --url="https://vdo.ninja/?push=mystream&ss=Discord&autostart"

Global Hotkeys

With elevated privileges enabled, Electron Capture supports global hotkeys:

  • CTRL+M - Mute/unmute microphone
  • Custom hotkeys can be set in VDO.Ninja Settings > User > Global Hotkey

Clean Video Output

For YouTube, Twitch, and other sites, Electron Capture can show just the video without overlays:

  1. Right-click on the page
  2. Select Clean Video Output
  3. The video fills the entire window

Bonus: YouTube videos get automatic ad-skipping!

Recommended URL Parameters

  • &scene - Optimized for OBS window capture
  • &codec=h264 - Force H.264 codec
  • &bitrate=6000 - Set target bitrate (kbps)
  • &quality=2 - Higher quality setting
  • &nocursor - Hide cursor in browser
