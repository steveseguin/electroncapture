Audio Setup

Configure virtual audio cables and per-window output routing.

Why Virtual Audio Cables?

Virtual audio cables let you route audio from Electron Capture into OBS or other applications as a separate audio source. This gives you independent volume control and the ability to apply filters.

Recommended Software

Windows

  • VB-Cable (free/donationware) - Simple, one virtual cable
  • Virtual Audio Cable (VAC) (paid) - Multiple cables, more features

macOS

  • BlackHole (free) - Open source, 2 or 16 channels
  • Loopback (paid) - Professional-grade, excellent UI

Changing Audio Output Device

To route audio to a virtual cable:

  1. Right-click in the Electron Capture window
  2. Select Audio Output Device
  3. Choose your virtual audio cable from the list

Note: You need to enable elevated privileges (right-click > Elevate Privileges) to change audio output devices.

Capturing in OBS

Once audio is routed to a virtual cable:

  1. In OBS, add an Audio Input Capture source
  2. Select your virtual audio cable as the device
  3. Audio from Electron Capture now appears as a separate track

Alternative: Application Audio Capture

Newer versions of OBS can capture application audio directly without virtual cables:

  1. Add an Application Audio Capture source
  2. Select Electron Capture from the application list

Split Audio/Video Capture

You can capture video with Electron Capture and audio with OBS Browser Source:

# Electron Capture - video only
elecap.exe --url="https://vdo.ninja/?view=abc123&noaudio"

# OBS Browser Source - audio only
https://vdo.ninja/?view=abc123&novideo

Tip: This setup may have slight audio/video sync differences, but is usually unnoticeable.

Multiple Audio Sources

For productions with multiple guests, each Electron Capture window can output to a different virtual cable:

  • Guest 1 → VB-Cable A → OBS Audio Track 1
  • Guest 2 → VB-Cable B → OBS Audio Track 2
  • Guest 3 → VB-Cable C → OBS Audio Track 3

This gives you independent audio control and multi-track recording.
