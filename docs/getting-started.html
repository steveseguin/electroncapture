Getting Started

Download, install, and run your first capture session in minutes.

Download

Visit the GitHub Releases page to download the latest version for your platform:

  • Windows: Choose the installer (.exe) or portable version (elecap.exe)
  • macOS: Download the .dmg file
  • Linux: AppImage files are available, or build from source

Tip: The portable Windows version is easier to use from the command line or batch files.

First Launch

When you first open Electron Capture, you'll see a minimal window with a URL bar. Here's how to get started:

  1. Enter a URL (e.g., https://vdo.ninja/) in the address bar
  2. Press Enter to load the page
  3. Right-click anywhere to access the context menu with options

Basic Controls

Right-click anywhere in the window to access the context menu:

  • Pin on Top: Keep the window above other applications
  • Change Resolution: Resize to common streaming resolutions
  • Elevate Privileges: Enable advanced features like screen sharing
  • Audio Output: Select which audio device to use
  • Close: Exit the application

Window Capture with OBS

To capture Electron Capture in OBS:

  1. Add a Window Capture source in OBS
  2. Select the Electron Capture window from the dropdown
  3. The window appears frameless - no browser chrome or address bar

Pro Tip: Use --title="My Stream" to set a custom window title, making it easier to identify in OBS.

Need audio? Window Capture in OBS captures video only. On Windows, Electron Capture can capture audio directly from applications - no virtual cables needed. See our Audio Capture Guide for details.
